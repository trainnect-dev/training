Sample Multiple Choice
  1. What format does OneLake use to store data?
    a) JSON b) Delta Parquet c) XML d) Avro
    Answer: b
  2. Which of the following is used to apply row-level permissions in Power BI?
    a) Audit Policy b) Sensitivity Labels c) RLS d) Power Query
    Answer: c
  3. What Fabric tool is used to create pipelines?
    a) Notebooks b) Dataflows Gen2 c) Azure Monitor d) Data Factory
    Answer: d
